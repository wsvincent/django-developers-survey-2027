# Contributing Guidelines

This repo is designed to gather community feedback on the 2026 Django Survey and suggest updates/changes for 2027. Please raise an Issue rather than submit a Pull Request for any changes.

You can [see the 2026 Django Survey here](https://surveys.jetbrains.com/s3/wb-django-developers-survey-2026).
